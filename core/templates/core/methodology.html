🔬 Methodology & Technology Stack

About PhishGuard AI

PhishGuard AI is an adaptive phishing training platform that uses machine learning to provide personalized phishing awareness training. Our system adapts to your skill level and identifies your weak areas to recommend targeted training.

📋 Adaptive Assessment

How it Works:

  • Initial baseline quiz with 10 random scenarios
  • Difficulty levels: Easy, Medium, Hard
  • System adapts scenarios based on your performance
  • Real-time response time tracking
  • Personalized recommendations based on weaknesses

Your skill level is continuously updated as you practice.

🤖 Random Forest Classifier

Algorithm Details:

  • Model: RandomForestClassifier (100 estimators)
  • Purpose: Classify user skill level
  • Features Analyzed:
    • Overall accuracy percentage
    • Average response time
    • Total attempts
    • Hard scenario accuracy
    • Medium scenario accuracy
  • Output: Beginner → Intermediate → Advanced
⚡ Adaptive Training Engine

How Adaptivity Works:

  • Beginner: Focuses on obvious phishing indicators (urgency, suspicious senders)
  • Intermediate: Increases difficulty; realistic formatting, better social engineering
  • Advanced: Complex scenarios with subtle indicators and sophisticated attacks
  • System recommends next difficulty based on skill classification
  • Recommendations target identified weaknesses

Your training path is customized for maximum learning impact.

🔍 Isolation Forest Anomaly Detection

Anomaly Detection Purpose:

  • Model: IsolationForest (contamination=0.05)
  • Detects unusual user behavior patterns
  • Features Monitored:
    • Accuracy trends
    • Response time consistency
    • Behavioral consistency
  • Alerts on potential account compromise
  • Flagged for security review
